How to get a token

  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic) or Fine-grained token
  3. Select the repo scope (add read:org to include organizations)
  4. Copy the generated token and paste it below (starts with ghp_ for classic or github_pat_ for fine-grained)
