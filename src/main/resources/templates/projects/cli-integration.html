CLI Integration

Run fast and secure scans on your local machine or build server.

This step requires a technical user
1

Install OsWL-cli

Run the following command from a local development Terminal or CI machine:

curl -H 'Cache-Control: no-cache' https://raw.githubusercontent.com/oswl/cli/master/install.sh | bash
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/oswl/cli/master/install.ps1'))
2

Set your API Key

Set your API key. You can create and manage API keys in Settings.

3

Run your first scan

In your project directory, run:

OSWL_API_KEY=your_api_key oswl analyze
