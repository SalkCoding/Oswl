CLI Integration

Run fast and secure scans on your local machine or build server.

API keys are project-scoped — issue one from the project settings before scanning
1

Install OsWL-cli

Run the following command from your Terminal or CI machine:

curl -fsSL https://raw.githubusercontent.com/SalkCoding/Oswl/develop/src/main/resources/static/scripts/install.sh | bash
iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/SalkCoding/Oswl/develop/src/main/resources/static/scripts/install.ps1'))
2

Get an API Key

Open the target project → Settings → CLI Keys tab → issue a new key. Each key is linked to exactly one project and identifies the scan target automatically. No other configuration is stored on your machine.

Keys can be revoked at any time from the same settings page. For CI/CD pipelines, store the key in an environment variable OSWL_API_KEY instead of passing it on the command line.
