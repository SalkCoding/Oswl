CLI Integration

Run fast and secure scans on your local machine or build server.

This step requires a technical user
1

Install OsWL-cli

Run the following command from your Terminal or CI machine:

curl -fsSL https://raw.githubusercontent.com/SalkCoding/Oswl/develop/src/main/resources/static/scripts/install.sh | bash
iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/SalkCoding/Oswl/develop/src/main/resources/static/scripts/install.ps1'))
2

Authenticate

Register your API key with the CLI. Run this once per machine — credentials are saved to ~/.oswl/config.

API keys are issued by an admin under Settings → CLI API Keys. Copy your key and use it in the command below.

3

Run your first scan

Navigate to your project directory and run:

cd /your/project && oswl scan
cd C:\your\project; oswl scan
